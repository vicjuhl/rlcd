# rlcd
Searches graph space with RL to find graphs which are likely to describe the process generating the data. The code is the implementation of the university project submission ``Causal Discovery on i.i.d. LiNGAM Data with Reinforcement Learning'', submitted on January 31st, 2026. Please contact victorkjellerupjuhl@me.com if you are interested in the pdf.

With conda installed, please do the following to install environment.

    conda env create -f environment.yml

To update existing conda environment based on file:

    conda env update -f environment.yml --prune
